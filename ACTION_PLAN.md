## ğŸ’¡ Pro Tips

1. **Image Quality Matters**
   - Low quality images make site look unprofessional
   - Invest time in good photos
   - Real team members look best

2. **Sizing is Critical**
   - Use exact specifications
   - Team images: 500x500 (square)
   - Hero: 1920x1080 minimum
   - Use free tools to resize: Pixlr, Canva

3. **File Size Optimization**
   - Compress images for faster loading
   - Use TinyPNG or Compressor.io
   - Aim for: Hero <500KB, Team <150KB each

4. **Naming is Exact**
   - Case-sensitive (lowercase recommended)
   - No spaces in filenames
   - Use hyphens: `team-sarah.jpg` (not `Team Sarah.jpg`)

5. **Backup Original**
   - Keep original images
   - Edit copies for web use
   - Good practice for future updates
