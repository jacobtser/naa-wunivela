## ğŸ“ Understanding the Code

### How It Works (Simple Version)

1. **Page Loads**
   - HTML carousel structure appears
   - JavaScript reads your 6 products
   - Creates carousel slides from products

2. **Autoplay Starts**
   - Timer set for 7 seconds
   - Every 7 seconds, shows next product
   - After last product, loops to first

3. **User Interaction**
   - Visitor clicks Previous/Next button
   - Carousel jumps to that product
   - Timer resets for next autoplay
   - Continues rotation

4. **Smooth Transitions**
   - Current product fades out
   - Next product fades in
   - Takes 0.8 seconds
   - CSS handles animation (smooth)

### Files Involved

| File | What It Does |
|------|-------------|
| `index.html` | Defines carousel HTML structure |
| `css/styles.css` | Colors, sizes, and styling |
| `css/responsive.css` | Mobile/tablet adjustments |
| `js/main.js` | Autoplay, transitions, interactions |

## ğŸ› Troubleshooting

### Carousel Not Showing
**Check**: Is `<div id="productsCarousel"></div>` in HTML?  
**Fix**: Open `index.html`, search for "productsCarousel", should be in products section.

### Autoplay Not Working
**Check**: Is JavaScript enabled? (Usually is)  
**Fix**: Open DevTools (F12), check console for errors.

### Wrong Speed
**Check**: Look for `7000` in `js/main.js`  
**Fix**: Change the number (milliseconds, so 7000 = 7 seconds).

### Mobile Layout Broken
**Check**: Responsive CSS media queries in `responsive.css`  
**Fix**: Test in DevTools mobile view, check breakpoints match.

### Can't Find Where to Change Something
**Fix**: Search the documentation files for that feature.
